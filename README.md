## üéØ Motivation

The financial sector generates massive volumes of complex and dynamic data, making traditional rule-based systems insufficient for modern financial decision-making. Artificial Intelligence offers powerful tools for automation, prediction, and optimization; however, challenges such as transparency, bias, data privacy, and regulation remain unresolved. This project is motivated by the need to study existing research in a structured manner and highlight how AI can be responsibly adopted in finance.
